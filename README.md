## Overview
The MCA Referee Test is an interactive, offline-ready multiple-choice quiz designed to assess referee knowledge of the Laws of the Game (LOTG). It consists of **10 questions** to be completed in **15 minutes**. The test simulates real-match scenarios to evaluate decision-making and understanding of the rules.

## Features
- **Timed Quiz:** 15-minute countdown with progress and time bar.
- **Multiple-Choice Questions:** 10 scenario-based questions.
- **Interactive UI:** Professional, clean interface for easy use.
- **Results Export:** Download results as CSV or HTML report.
- **Offline Ready:** Fully functional without an internet connection.

## Usage
1. Launch the test by clicking **Start Test**.
2. Answer all 10 questions using the **Next** and **Previous** buttons.
3. Monitor the countdown timer; the test automatically ends when time runs out.
4. Click **Submit** to finish and view your results.
5. Export your results using the **Download CSV** or **Download HTML Report** buttons.

## Customization
- **Questions:** Edit `js/script.js` to add, remove, or modify questions.
- **Styles:** Customize `css/style.css` to change colors, fonts, or layout.
- **Timer:** Adjust the `totalTime` variable in `script.js` to change quiz duration.
